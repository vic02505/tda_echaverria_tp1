Algoritmo MinPalindromos(S)

    n := longitud(S)

    Si n = 0 entonces
        Retornar 0
    FinSi

    Crear matriz pal de n x n inicializada en Falso

    Para i := 0 hasta n - 1 hacer
        pal[i][i] := Verdadero
    FinPara

    Para largo := 2 hasta n hacer
        Para i := 0 hasta n - largo hacer
            j := i + largo - 1

            Si S[i] = S[j] y (largo = 2 o pal[i + 1][j - 1]) entonces
                pal[i][j] := Verdadero
            FinSi
        FinPara
    FinPara

    Crear vector dp de tamano n inicializado en 0

    Para i := 0 hasta n - 1 hacer

        Si pal[0][i] entonces
            dp[i] := 1
        Sino
            dp[i] := infinito

            Para j := 1 hasta i hacer
                Si pal[j][i] entonces
                    dp[i] := minimo(dp[i], dp[j - 1] + 1)
                FinSi
            FinPara
        FinSi

    FinPara

    Retornar dp[n - 1]

FinAlgoritmo
